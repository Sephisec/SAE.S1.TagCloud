Tutoriel

Choisir les textes

Récuperez le contenu à analyser au format .txt.
Vous pouvez faire votre choix sur Wikisource par exemple.

Déplacer les fichiers

Déplacez ensuite les fichiers obtenus dans le répertoire "txt" de l'outil.
Le nom de votre fichier doit être le nom de l'oeuvre à analyser. L'arborescence du répertoire devrait ressembler à l'image ci-contre.

Executer tagcloud.exe

Ouvrez le terminal Linux et entrez la commande "mono tagcloud.exe" ou éxecutez-le comme vous le faites habituellement.
Attendez que le programme s'execute.
Il vous sera demandé d'entrer le prénom et le nom de l'auteur.

  • juge
  • violences
  • sarkozy
  • police
  • peut
  • tout
  • loi
  • être
  • ans
  • nicolas
  • ministre
  • peine
  • plus
  • délinquance
  • prison

Se rendre sur l'interface web

Ouvrez index.html et cliquez sur "C'est parti".
Les 15 mots les plus récurrents seront affichés sous forme de nuage de mots.
Ci-contre le résultat de l'analyse de l'exemple.
